# waffle_L

Учебный телеграм-бот на Python (aiogram 3), который составляет план питания.

@waffle_with_losos_bot | https://t.me/waffle_with_losos_bot

## Что делает бот

- Спрашивает у пользователя пол, возраст, частоту готовки, приоритеты, нужные приёмы пищи, рост, вес и количество персон
- Считает норму калорий по этим данным
- Просит LLM (через OpenRouter) составить план питания на несколько дней с рецептами
- Позволяет посмотреть план по дням, заменить отдельное блюдо и сгенерировать список покупок
- Даёт изменить любой параметр анкеты в настройках

## Стек

Python, aiogram 3, OpenAI SDK (используется для запросов к OpenRouter), хранение данных в обычных JSON-файлах.

## Структура

```
src/
├── bot.py             # вся логика бота
├── recipes.json       # тестовая база рецептов
├── users.json         # профили и планы пользователей
├── secret.py          # токены
└── requirements.txt
```

## Запуск

1. Клонировать репозиторий и перейти в папку `src`
2. Установить зависимости:
   ```
   pip install -r requirements.txt
   ```
3. В `secret.py` вставить свои токены — Telegram-бота и OpenRouter
4. В `bot.py` в функции `call_llm` (строка ~693) указать название модели, например `nvidia/nemotron-3-super-120b-a12b:free`
5. Запустить:
   ```
   python bot.py
   ```

## Примечание

`recipes.json` — тестовые данные для демонстрации, их можно заменить своими. Проект учебный, хранение в JSON подходит только для небольшой нагрузки.
